# Data access

The clinical data and derived datasets used by this project are not included in
the GitHub repository.

The repository contains code, configuration templates, and tests only. The data
will be distributed separately through PhysioNet under the access controls and
data-use agreement applicable to the source dataset. A project URL and versioned
data citation will be added here after the PhysioNet publication is approved.

Do not open a GitHub issue requesting copies of restricted data. Users must
obtain access through the official PhysioNet project page and comply with all
credentialing, training, and data-use requirements.

Expected local layout after authorized download:

```text
data/
  cleaned/
  stream/
  predictions/
  metrics/
  audit/
```

The entire `data/` directory is intentionally ignored by Git.
